## Données des tests

* Les jeux tests sont dans le dossier `${projectDir}/tests/data`
* Les modèles de données tests utilisés sont des modèles externes (ex : https://www.geoportail-urbanisme.gouv.fr/standard/cnig_SUP_PM3_2016.json)
